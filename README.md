注意: 本规则可能会收集一些广告数据,用于改善规则中存在的问题

更新记录: CloudGate规则 更新记录 列表

规则地址

备用线路: https://Config.daoapp.io/Proxy/Shadowrocket.php

更新规则

推荐使用「在线导入」方式更新规则文件

使用方法

第一步:「Shadowrocket」>「配置」>「+」>「主用线路/备用线路」>「下载」

第二步:「Shadowrocket」>「配置」>「本地文件」>「规则」>「使用配置」
